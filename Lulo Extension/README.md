# Lulo - AI Assistant Chrome Extension

Your AI team, working in parallel. Browser automation and task assistance powered by Gemini AI.

## Features

- **AI Chat Assistant**: Get help with any task through natural conversation
- **Browser Automation**: Click elements, fill forms, extract content from pages
- **Side Panel Interface**: Full chat experience without leaving your current page
- **Screenshot Capture**: Capture visible tab screenshots for reference
- **Smart Actions**: Quick access to common automation tasks

## Installation

1. Visit the Chrome Web Store
2. Click "Add to Chrome"
3. Click the Lulo icon in your toolbar to get started!

## Usage

### Popup
Click the Lulo icon for quick task entry. Type your request and press Enter or click "Start Task".

### Side Panel
Click "Open Full Panel" for a complete chat experience. The side panel stays open while you browse.
